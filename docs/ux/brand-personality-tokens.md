# Brand Personality Tokens

## Chosen Personality Tokens

- Primary: Calm Utility
- Secondary: Editorial Precision
- Secondary: Warm Guidance

## Translation Notes

- Typography: confident hierarchy, restrained body copy, readable density
- Color: earthy or grounded surfaces with selective high-signal accents
- Surfaces: composed, layered, low-noise containers
- Motion: subtle orientation and state feedback
- Interaction tone: supportive, focused, never loud
