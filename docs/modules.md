# Modules

Nest is composed of several functional modules.

---

# Task System

Manages tasks and task lists.

Supports:

- priorities
- deadlines
- projects
- life areas

---

# Habit System

Tracks recurring behaviors.

Supports logging and progress tracking.

---

# Routine System

Defines sequences of actions performed regularly.

Examples:

- morning routine
- work day routine

---

# Goals System

Manages long-term objectives.

Goals contain targets.

---

# Journal

Allows users to record thoughts, reflections, and experiences.

---

# Life Areas

Categorizes life activities.

Provides balance analysis.

---

# Lists

General purpose lists such as:

- shopping lists
- idea lists
- checklists
