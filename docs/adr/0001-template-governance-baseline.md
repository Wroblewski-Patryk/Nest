# ADR 0001: Template Governance Baseline

- Status: accepted
- Date: YYYY-MM-DD

## Context
Multiple projects share one template and need consistent agent behavior.

## Decision
Use this template as the canonical baseline for:
- planning and task orchestration
- documentation and review standards
- UX evidence policy
- subagent delegation policy

## Consequences
- easier cross-project onboarding
- predictable execution and review quality
- explicit tradeoff: template maintenance overhead
